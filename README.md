# Flask Hotel Booking Dashboard
A web-based hotel booking management system built with Python and Flask.

## Features
- Add hotel bookings with guest name, room number, check-in and check-out dates
- View all bookings in a table
- Delete bookings
- Connected to SQLite database

## Built with
- Python
- Flask
- SQLite3
- HTML

## How to run
1. pip install flask
2. python app.py
3. Open http://127.0.0.1:5000
